# Sibling Showdown

A private two-person weight-loss competition app for friendly daily check-ins, meal prep, hydration, workouts, and playful mascot rivalry.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Local mock data for the MVP
- Ready for Vercel deployment later

## Run Locally

```bash
npm install
npm run dev
```

## Beginner Tip

Start by reading `src/lib/types.ts`, then `src/lib/mock-data.ts`, then `src/app/page.tsx`. Those three files show the shape of the app: what data exists, what sample data is loaded, and how the home screen chooses morning vs evening.
